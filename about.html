About this Blog

This blog strives to

  • cover interesting things in Python and other programming languages
  • introduce readers to tools that are super useful to know exist
  • strike the same balance between opinion and fact that can be found in The Hitchhiker's Guide to Python. Opinionated content makes it quicker to skip to the meat of programming when you start out as a developer

If you want to hear about it when I post something new, you can use the RSS feed or follow me on Twitter.
